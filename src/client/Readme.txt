To install and use the desktop-app :

1. launch the installer file (P2P Secure File Transfer-1.0.0 Setup.exe)
2. The program will install, test by launching a first time, then restart and launch a second time. Then, it is ready to be used.
3. Ping the program to the task bar. Thus, you can launch it without reinstalling.
4. As an alternative, you might create a shortcut of the installed executable at "C:\Users\(Your_Username)\AppData\Local\travail_de_bachelor\app-1.0.0\P2P Secure File Transfer.exe"
